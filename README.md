# C# ASP.NET + React Native Application

## Usage

## Prerequisites

```
- .NET SDK
- NodeJS
- Docker
```

### Environment Variables

```dotenv
# db config
MYSQL_USER=
MYSQL_PASSWORD=
MYSQL_DATABASE=
```

### Offline Mode

Backend is available in `5016` port, and Frontend in `8081`

```
# Running services
docker-compose up
```

## Documentation

Endpoints are available in `swagger` ASP.NET interface.

```
> Store endpoints
  - Get stores
  - Get store
  - Post store
  - Put store
  - Delete store
```
